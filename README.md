# Prithvi Buildtech

Оферта:

Статический сайт компании Prithvi Buildtech, реализованный на HTML/CSS/JS в рамках небольшого проекта.

## Структура проекта

- `index.html` - главная страница
- `about.html` - о компании
- `service.html` - услуги
- `contact_us.html` - контакты
- `privacy-policy.html` - политика конфиденциальности
- `terms-conditions.html` - условия использования
- `thanks.html` - страница благодарности
- `style.css` - стили
- `script.js` - скрипты
- `icons/`, `images/`, `fonts/` - ассеты

## Запуск

1. Откройте `index.html` в браузере.
2. Для локального сервера (рекомендуется):
   - `python3 -m http.server 8000`
   - в браузере: `http://localhost:8000`

## Команда

- html / css / js
- адаптивная верстка
- простая форма контактов
